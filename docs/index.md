# AI Governance Tool Documentation

Welcome to the AI Governance Tool documentation. This tool helps organizations assess and maintain compliance with ISO 42001 AI Management System requirements.

## Overview

The AI Governance Tool provides:

- Automated compliance checking against ISO 42001 standards
- Pattern matching for key compliance terms
- Proximity analysis of related concepts
- Comprehensive reporting in PDF format
- Historical compliance tracking

## Quick Start

1. Install the package:
```bash
pip install -r requirements.txt
```

2. Run a compliance check:
```bash
python main.py
```

3. View the generated report in `compliance_report.pdf`

## Key Features

- **Pattern Matching**: Sophisticated regex-based detection of compliance terms
- **Proximity Analysis**: Understanding relationships between compliance concepts
- **Rich Reporting**: Detailed PDF reports with charts and trends
- **Historical Tracking**: SQLite database for compliance history
- **Comprehensive Testing**: Full test coverage
